# ADR-0001: GitHub und Markdown als Single Source of Truth

Status: Ersetzt durch [`ADR-0003`](adr-0003-github-als-architecture-workspace.md)
Datum: 2026-07-14
Owner: Hauptstammleitung

## Historischer Kontext

Die erste Repository-Version verstand GitHub und Markdown primär als Single
Source of Truth für eine versionierte RRLA-Dokumentation. Das sollte lose
Notizen und konkurrierende Dokumentstände vermeiden.

## Frühere Entscheidung

RRLA wird in GitHub als Markdown-Dokumentation gepflegt. Zielarchitektur, offene
Fragen, Entscheidungen, Roadmap, Templates und Glossar werden dort zentral
gehalten.

## Warum ersetzt?

Die Formulierung „Single Source of Truth“ legte den Schwerpunkt zu stark auf
fertige Inhalte. RRLA wird inzwischen als gemeinsamer Denk- und
Architekturprozess verstanden. GitHub bleibt der zentrale Ort, dient aber vor
allem als Architecture Workspace und Organisationsgedächtnis.
